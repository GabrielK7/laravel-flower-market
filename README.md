# FlowerShop (Laravel)

## Features
- Login / Logout
- CRUD products
- Categories
- Search + filter
- Pagination

## Tech stack
- Laravel 12
- MySQL
- Bootstrap 5

## Installation
1. git clone ...
2. composer install
3. cp .env.example .env
4. php artisan key:generate
5. php artisan migrate
6. php artisan serve
